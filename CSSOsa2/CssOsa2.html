grid-template-columns
grid-template-rows

grid-template-columns komennolla määrritellään montako ja minkän kokoista "pylvästä" sivulle luodaan.
grid-template-rows komennolla määrritellään montako ja minkän kokoista "riviä" luodaan.

grid-template-areas

tällä on helppo luoda sivun layout.
grid-area: komennolla voit "itemin" asettaa nimetylle paikalle.
klikkaa napista nähdäksesi tämän sivun layout.

Pienemmän näytön layout:


.container{
display: grid;
grid-template-columns: 1fr;
grid-template-rows: auto;
gap: 20px;
grid-template-areas:
"header"
"content-1"
"content-2"
"content-3"
"content-4"
"content-5"
"content-6"
"footer";
}


Isomman näytön layout:


@media(min-width:600px){
.container{
display: grid;
grid-template-columns: 1fr 1fr;
grid-template-rows: auto auto;
gap:20px;
grid-template-areas:
"header header"
"content-1 content-2"
"content-3 content-4"
"content-5 content-6"
"footer footer";
}

column-gap/row-gap/gap

column-gap komennolla määritellään kuinka iso "rako" pylväiden välillä on.
row-gap komennolla taas määritellään kuinka iso "rako" rivien välillä on.
gap komento taas määrittää molemmat.

justify-self

justify-self komennolla voidaan "itemi" asetella "solun" sisällä vaaka tasossa.

justify-self: start;

Asettaa itemin "solun" vasemmalle reunalle

justify-self: end;

asettaa itemin "solun" oikeilla reunalle.

justify-self: center;

asettaa itemin "solun" keskelle.

justify-self: stretch

täyttää koko solun.

align-self

align-self komennolla voidaan "itemi" asetella "solun" sisällä pysty suunnassa.

justify-self: start;

Asettaa itemin "solun" yläreunaan

align-self: end;

asettaa itemin "solun" alareunaan.

align-self: center;

asettaa itemin "solun" keskelle.

align-self: stretch

täyttää koko solun.
